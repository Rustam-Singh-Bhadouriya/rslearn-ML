# Contributing to rslearn 🚀

Thank you for your interest in contributing to **rslearn**!

This project is designed to be a **beginner-friendly machine learning library**, so all contributions should follow simplicity, clarity, and ease of use.

---

## 🧠 Before You Start

Please understand the goal of this project:

- Make ML easier for beginners
- Reduce unnecessary complexity
- Provide clean and readable APIs
- Keep behavior predictable and consistent

---

## 🛠️ How You Can Contribute

You can contribute in the following ways:

- 🐛 Reporting bugs
- ✨ Suggesting new features
- 📚 Improving documentation
- 🔧 Fixing issues or adding enhancements
- 🧪 Writing tests or edge case checks

---

## 📌 Code Style Guidelines

Please follow these rules:

- Keep functions simple and readable
- Avoid unnecessary complexity
- Use meaningful commit messages (e.g., `feat:`, `fix:`, `refactor:`)
- Follow existing structure of the project
- Add comments only when necessary (code should be self-explanatory)

---
